# 観戦HQ '26 — FIFAワールドカップ2026 観戦コンパニオン

「寝不足上等。全104試合、見逃すな。」

FIFAワールドカップ2026(アメリカ・カナダ・メキシコ共催)を日本時間で完全攻略するためのオールインワン観戦Webアプリ。

## 起動

```bash
npm install
npm run dev   # http://localhost:5180
npm run build # 本番ビルド
```

## 機能

| タブ | 内容 |
|---|---|
| ホーム | 日本戦カウントダウン、今日・明日の試合、大会10大ストーリー、死の組 |
| 試合日程 | 全104試合(JST)。節/グループ/チーム/放送区分/日本戦/観戦予定/推しチームでフィルター。Googleカレンダーにワンクリック追加 |
| 日本代表 | 速報・26人メンバー・対戦相手スカウティング・キーマッチアップ・突破シナリオ・F組順位表 |
| 組&チーム | 12組順位表(結果から自動計算)+全48カ国図鑑(FIFAランク・注目選手・雑学) |
| ランキング | 得点/アシストランキング、得点王候補オッズ、優勝オッズ、懸かる記録、若手5傑 |
| 視聴ガイド | DAZN無料/独占/地上波の完全整理。プラン料金比較、無料で観る4つの方法 |
| マイ観戦 | 推しチーム登録、勝敗予想と的中率、観戦予定+睡眠プランナー、結果入力 |

## データ更新(大会進行に合わせて)

- **試合結果**: [src/data/results.ts](src/data/results.ts) の `officialResults` に追記(またはアプリ内「マイ観戦→結果入力」)。順位表・的中判定に自動反映。
- **得点/アシスト**: 同ファイルの `officialScorers` に追記。ランキングに自動反映。
- **決勝T組み合わせ**: 6/28確定後、[src/data/matches.ts](src/data/matches.ts) の `knockoutMatches` の `home`/`away` を更新。

## データの出典

2026年6月12日時点のFIFA公式・DAZN公式(2025-12-04/2026-05-30プレスリリース)・JFA公式発表・各種報道(ESPN, NHK, サッカーキング, ゲキサカ等)に基づくファンメイドの非公式アプリ。

- 放送区分(DAZN独占/無料/地上波)はDAZN公式試合スケジュール画像(2026年6月版)を転記
- 国旗画像: [flagcdn.com](https://flagcdn.com)
- ヒーロー画像: GPT Image 2(Codex CLI経由)で生成

## 技術スタック

Vite + React 19 + TypeScript + Tailwind CSS 3 / lucide-react / localStorage永続化(サーバー不要・完全静的)
